Whenever you add any content or change any settings you will need to re-save your own personal data.

Process/処理方法
1.Press the "Personal data" button
2.The data will automatically be saved to your browser
3.(Optional) Copy-Paste the data below to a backup file on your computer
1.「個人データ」ボタンを押す
2.データが自動的にブラウザに保存される
3.（任意）テキストボクスの内容を自分のコンピューターのバックアップファイルにコピーペーストする
●英語●Share●英語●●日本語●共有●日本語●
? Share data file/共有データファイル
Share data is used for sharing templates and manuals that you have created.
共有のデータは自分が作成したテンプレート・マニュアルを共有するときに使う。
Process　Share/共有処理方法
  1. Share the template/manual with your team for confirming that it is acceptable and that no one opose to add it to the master data
  2. If there are any issues, review the template/manual and start over at step 1, otherwise proceed with step 3
  3. Press the "Share data" button
  4. Select the templates/manuals that you want to share and press "Generate data"
  5. Copy paste all the text in the text box to a textfile on your computer
  6. Share the file with the person responsible to update the master data
  1. チーム内でテンプレート・マニュアルを共有して、問題が無いか・マスターデータに追加する反対が無いかを確認する
  2. もし問題があったら、問題点を修正してステップ１に戻る、他の場合にはステップ３で続く
  3. 「共有データ」ボタンを押す
  4. 共有したテンプレート・マニュアルを選択し、「データ作成」を押す
  5. テキストボクスの内容を自分のコンピューターのテキストファイルにコピーペーストする
  6. 作成したファイルをマスターデータを更新するスタッフに渡す
Process Load/ロード処理方法
  1. Copy paste the shared data in the text box below
  2. Press "Load data"
  1. 共有されたデータを下記のテキストボクスにコピーペーストする
  2. 「データロード」を押す

Master data is used for shared data that all users should use (official approved data)

  2. テキストボクスの内容をサーバーの「Master.txt」のテキストファイルにコピーペーストする

? Approval process/承認処理方法
  1. Press the "Approve" button to approve the data as master data (Should only be used if the data follows our current rules, in all other cases it is nessesary to get approval for the process beforehand)
  2. Press the "Approve (Private)" button to add the data as private data (Use this if further confirmation is needed)
  3. Press the "Reject" button to avoid any changes (Should be used if there is any problems with the suggested data that needs to be fixed)
  1. 「承認」ボタンを押すと、データがマスターデータとして追加されます（今のルールに従っているものにしか使えない、その他の場合には、先ず処理方法の承認が必要）
  2. 「承認（個人）」ボタンを押すと、個人データとして追加されます（もしまだ承認・確認が必要な場合には、こちらを使ってください）
  3. 「拒否」ボタンを押すと、何も変わらない（もし修正が必要な問題があったら、こちらを使ってください）
